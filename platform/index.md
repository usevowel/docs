# Platform

The hosted platform is coming soon.

## Current Launch Status

The initial open-source launch is focused on the self-hosted stack.

Today, use:

- [Client](/guide/getting-started) for SDK integration
- [Self-Hosted](/self-hosted/) for deployment and operations

## What Is Coming Later

The hosted platform will cover:

- hosted app creation and configuration
- testing and debugging in a managed control plane
- hosted billing and organization workflows
- `appId`-based setup
- `vowelbot`
- hosted realtime API features
