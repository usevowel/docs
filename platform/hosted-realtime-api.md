# Hosted Realtime API

Hosted realtime API support is coming soon.

## What To Use Today

Until the hosted realtime API is available, the recommended paths are:

- the self-hosted stack
- token-based client integrations backed by your own backend
- the hosted platform flow with `appId`, once that launches
