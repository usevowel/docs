# voweldoc

## Overview

Present a modal dialog that allows users to enter either a url/vowel appid combo or JWT (set by env var) which will be stored in local storage to populate the vowel client
This key / token can come from either a vowel [SAAS subscription](https://vowel.to) or [Self-hosted](https://docs.vowel.to/self-hosted) instance.

Then the user can navigate with voice and ask quesitons about the docs.
